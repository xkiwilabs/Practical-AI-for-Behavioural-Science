Course Repository (Essential)

All course materials live in a public GitHub repository. This is your central hub for the course — bookmark it:

github.com/xkiwilabs/Current-Advances-in-Psychological-Methods-and-Analyses-Repo

The repository contains:

  • Weekly companion readings — the required reading for each week
  • Lecture slides — viewable in your browser
  • Challenge lab briefs and starter notebooks — what you'll work on in lab weeks
  • Datasets — for all lab challenges
  • Setup guide and scripts — everything you need to install Python, VS Code, and course packages
  • Resource guides — AI tools, prompt engineering, HTML slides, Jupyter notebooks, and more
  • Assessment details — rubrics and instructions for all three assessments

You don't need to know how to use Git or GitHub — just open the link in your browser and navigate the folders. Think of it like a shared Google Drive, but for code and course materials.

Unit Readings

Each lecture week has two types of reading material in the course repository:

  • Companion reading (required): The README.md file in each week's folder is the main companion reading. It covers the lecture topics in detail and at your own pace. Read it before or after the lecture — it's the required reading for each week.
  • Additional readings (optional): The readings.md file in each week's folder lists suggested and optional papers. These cover and extend material from the lectures. Optional readings are also provided to help students identify research papers for the presentation and written assignment.

Technology Needs

Laptop

Bring a laptop to class if you have one. During lab weeks, you will work in small groups on coding challenges. If you don't have a laptop, you can share with your group members — at least one laptop per group is required.

Software Setup

You will need:

  • Python with Jupyter Notebooks for coding challenges
  • VS Code as your code editor: code.visualstudio.com (free)
  • An LLM assistant (ChatGPT, Claude, Gemini, or similar) for coding help and the written assignment

Detailed setup instructions are in the course repository: setup/getting-started.md. Please complete the setup before Week 2. Part of the Week 2 class will be dedicated to troubleshooting any issues.

All software used in this course is free or has free access for students.

LLM Access

You will use LLM assistants throughout this course for coding help, debugging, and the written assignment. All of these have free tiers:

  • ChatGPT: chat.openai.com
  • Claude: claude.ai
  • Gemini: gemini.google.com
  • GitHub Copilot: github.com/features/copilot (free for students via the GitHub Student Developer Pack)

For a full overview of available AI tools, see the AI Tools Guide in the course repository.

Requirement: You must be able to copy or export your chat history for the written assignment.

GitHub Account

Sign up for a free GitHub account at github.com. We recommend using a personal email address (not your MQ email) — your MQ email will stop working after you graduate, but your GitHub account will be useful for years to come.

Once you have an account:

  1. Add your @students.mq.edu.au email in Settings → Emails

General Technology Requirements

Computers and access to the Internet are essential for participating in this course. You must use them to access content saved on iLearn and complete online activities and assessments.

Technical difficulties are not grounds for special consideration, and in most cases, extensions to deadlines will not be granted. If you want to avoid losing any unnecessary marks, you should take responsibility for your computing. You must factor in potential computer and Internet problems when aiming to meet a deadline. This means regularly saving backup copies of your work and submitting early to allow for unforeseen issues arising.

If you have planned accordingly and problems arise, you can complete your assessments online using a computer in one of Macquarie University's many computing labs. As such, technical problems are not grounds for special consideration for any assessment task.

If you encounter problems while being diligent, complete the steps outlined below:

  1. Take a screenshot of any error message or similar message that comes up on the screen. You must show evidence that something went wrong.
  2. Contact technical support:
    • For iLearn problems, put in a OneHelp ticket request by emailing: ilearn.help@mq.edu.au, including the screenshot of the error message as an attachment and outlining what went wrong.
  3. If technical support does not resolve the issue, email the Convenor and include your correspondence with the relevant technical support teams.

Course Datasets

Datasets for all lab challenges are in the course repository, inside each week's data/ folder. You do not need to find your own data for the group challenges.

For the Written Assignment

You will need to find a research paper that uses ML/AI methods in psychological or cognitive science research. Good places to search:

  • Google Scholar
  • PubMed (for clinical/health psychology)
  • PsycINFO (via library)
  • Semantic Scholar

Tip: Use your LLM assistant to help find relevant papers, but verify that the papers actually exist and say what the LLM claims.

Email Use

It is University policy that the university-issued email account will be used for official University communication. All students are required to access their University account frequently. Only contact Macquarie University staff (including tutors) using your official MQ student's account because this is one method used to verify your identity.
